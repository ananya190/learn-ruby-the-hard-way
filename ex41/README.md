# RUBY OOPS 

## Word drills

+ class: tell ruby to make a new "type" of thing
+ object: the most basic type of thing and any instance of some thing
+ instance: what you get when you tell Ruby to create a class
+ def: to define a function in a class

+ @: variable for the instance/object being used
+ inheritance: the concept that one class can inherit traits from another
+ composition: the concept that a class can be composed of other classes 
+ attribute: a property classes have that are from composition and are usually 
variables
+ is-a: phrase to say that something inherits from another, like 
"salmon" is-a "fish"
+ has-a: phrase to say that something is composed of other things, like
"a salmon has-a mouth"

## Phrase drills

+ class X < y - make a class X that is-a Y
+ class X: def initialize(J) - class X has-a initialize that takes parameter J
+ class X: def M(J) - class X has-a function named M that takes parameter J
+ foo = X.new() - set foo to an instance of class X
+ foo.M(J) - from foo, get the function M and call it with parameter J
+ foo.K = Q - assign attribute K of foo the value Q
